Welcome to Link Cleaner

Link Cleaner is a web app that removes tracking code, search parameters, and other data from URL links. It's a great way to shorten links for text chats, sharing in social media posts, or embedding in documents and articles.

Link Cleaner is created by Corbin Davenport and open-source software.

Share through everything

Thanks to the Web Share API, Link Cleaner can share cleaned links directly to other applications. If you install Link Cleaner as an application, it can also appear in the share menu in your other applications. This works best on Google Chrome for Android, Microsoft Edge on Windows, and other browsers that fully support the Web Share API.

Mac, iPhone, and iPad integration

The official Apple Shortcut can add Link Cleaner to your iPhone or Mac's share menu. It's a great way to quickly filter links before using or sharing them elsewhere. You can also add Link Cleaner to your home screen or Dock.

Perfect for publishers

Link Cleaner can help you clean up links before using them in articles, documents, podcasts, or other published content. If you have an Amazon store affilate ID, add it in the settings page and Link Cleaner will apply it to all cleaned Amazon links.

Selective link processing

Link Cleaner removes tracking and junk data from links, while keeping the parameters required for the page to load. For example, YouTube video links will retain the watch?v= video ID, but not the ?si= parameter for tracking who shared the link.

Custom workflows and automations

Link Cleaner can be used in scripts or other automation tools, using the custom share URL. There's also an official bookmarklet for easy link cleaning on desktop web browsers.

Easy bulk cleaning

You can use Bulk Mode to quickly clean many links in one click, using the same settings and per-site overrides as the regular Link Cleaner mode.

Privacy first

Link Cleaner creates cleaned links and QR codes locally in your web browser, without sending the URLs to an external server. This fork removes all analytics and tracking.

Open-source software

Link Cleaner is an open-source software project on GitHub under the GPLv3 license. You can check out the code, suggest features or bugs, or create your own version.

Install Link Cleaner

This will install Link Cleaner on your device as an application.

On Chrome for Android and other supported platforms, this will add Link Cleaner to your system's share menu so it can recieve links.

In the Safari web browser, select the button, then select Add to Home Screen or Add to Dock to install Link Cleaner.

On Mac, you need macOS Sonoma 14.0 or higher to install web apps through Safari.

This shortcut adds Link Cleaner to the Share Sheet on iPhone, iPad, and Mac devices.

Install Apple Shortcut

You can use the Shortcut as a filter while sharing links. Instead of selecting the app or contact directly, select Link Cleaner first, then use the Share button in the Link Cleaner window.

The bookmarklet cleans your current page URL in a Link Cleaner popup window. Drag the below link to your bookmarks bar to install it.

Link Cleaner

If your bookmarks bar is not visible, try using the Ctrl+Shift+B (Windows) or Cmd+Shift+B (Mac) keyboard shortcut.

You can use Link Cleaner in custom worfklows and automations with the share URL. This is an example using the https://example.com page:

https://linkcleaner.app/?url=https%3A%2F%2Fexample.com

The only required parameter is url. The URL should be encoded with encodeURIComponent() or another similar function, or special characters like ? or & could break the URL.
